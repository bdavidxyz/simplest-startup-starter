Features

It's a classic amongst landing page sections : the section that list the features that your service provides

Robust & extensible. Simple design, implemented with jQuery and SASS. If you're not afraid by coding you can add any section you want easily.
Avoid technology lock.The generated webpage is only made of web front-end technologies : HTML5, CSS3, JS. You may use (but are not locked into) any back-end technology.
To showcase the feature, we use here the classic CSS pattern named "media object". The media object is an image to the left, with descriptive content to the right.
Notice here we use the same image 4 times, but usually you will pick one different image for each text.
